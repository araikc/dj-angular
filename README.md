# django-angular project: includes authentication

## Installation

* Fork this repository.
* `$ git clone git@github.com:<your username>/dj-angular.git`
* `$ cd dj-angular`
* `$ pip install -r requirements.txt`
* `$ npm install -g bower`
* `$ npm install`
* `$ bower install`
* `$ python manage.py migrate`
* `$ python manage.py runserver`
